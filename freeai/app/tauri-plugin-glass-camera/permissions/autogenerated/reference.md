## Default Permission

Allows connecting to the Meta glasses camera and capturing photos.

#### This default permission set includes the following:

- `allow-start-glass`
- `allow-stop-glass`
- `allow-capture-photo`
- `allow-start-registration`
- `allow-registration-state`
