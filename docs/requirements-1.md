# nukon 要件定義書v1

## 1. やりたいこと

Raspberry Piを用いたトイカメラで「ネットワークにつながるカメラ」という体験を実現する。また、「撮る体験」を最大化する物理ハードウェアの制作を行う。

## 2. 何をやるか

Raspberry Piを中核に据え、トイカメラを制作する。
このトイカメラは2つのモジュールからなる。

### メインモジュール

Raspberry Piと大型のバッテリー、タッチパネルとEVFを備えたメインのカメラモジュール。

以下のような構成を想定する。

- Raspberry Pi 5
- 64MP カメラモジュール
  - https://www.arducam.com/64mp-af-for-raspberry-pi.html
  - AF/プログラマブルなフォーカスが要件。
- EVFモジュール（Aliexpressに売っている）
  - アクセサリーシューにくっつけるようなのもある
  - 組み込んでも良い
- バッテリーモジュール（自作？）
  - 電源ボタンとか充電まわりが鬼門

### サブモジュール

矢印キーやシャッターボタン等と小型のシャッターボタンを備えたサブモジュール。

- 2段押しのできるシャッターボタン
  - ピエゾ素子とハプティックフィードバックか？
- 矢印キーがあると良い。
- バッテリーモジュール（自作？）
  - 電源ボタンとか充電まわりが鬼門
